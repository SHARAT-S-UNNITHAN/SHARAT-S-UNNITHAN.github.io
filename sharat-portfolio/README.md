# Sharat S Unnithan - Full-Stack Developer Portfolio

A modern, responsive portfolio website for a freelance full-stack developer featuring advanced animations, dark mode, and interactive elements.

## Features

### Design & Layout
- Modern, clean UI with glassmorphism elements
- Fully responsive design (mobile-first approach)
- Dark/Light mode toggle
- Advanced CSS animations and transitions
- Smooth scrolling and scroll animations
- Interactive hover effects

### Pages
1. **Home** - Hero section, services, pricing, testimonials
2. **About** - Bio, skills, experience timeline
3. **Projects** - Project showcase with filtering
4. **Contact** - Contact form with validation

### Functionality
- Dynamic text animations
- Project filtering by category
- Testimonial slider
- Form validation
- Back to top button
- Page transitions
- Skill progress bars
- Counter animations

### Technologies
- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript
- Google Fonts
- Font Awesome Icons

## Setup Instructions

1. Clone or download the project files
2. Replace placeholder content with your own:
   - Update personal information in HTML files
   - Replace placeholder images in `/assets/images/`
   - Customize colors in CSS variables
   - Update project details and links

3. For image placeholders, you can:
   - Use AI-generated images
   - Add your own photos
   - Use stock photos with proper licensing

## Customization
